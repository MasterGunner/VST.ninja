Desert Bus Chat

The Desert Bus community hangs around all year at irc.desertbus.org, come in for a chat!

During Desert Bus For Hope itself, we relocate to the "bunker", so as to avoid adding strain to the Desert Bus servers. You can find it over at irc.chatspike.net #DesertBusBunker.

If you would rather use a dedicated chat program, rather than the web clients linked above, we recommend HexChat. If you have trouble setting it up, hop into chat using the above web client link, and someone will be able to help you out.
