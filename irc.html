Desert Bus Chat

The Desert Bus Chat is a great community that exists year-round. Playing games, talking TV and movies, and always lending an ear to those who need one. Come on by, and feel free to join us.

If you would rather use a dedicated chat program, rather than the web clients linked below, we recommend HexChat. If you have trouble setting it up, hop into chat using the web client, and someone will be able to help you out.

Desert Bus IRC

This is the normal hangout of the Desert Bus community, you can find us here all while Desert Bus for Hope isn't going on: irc.desertbus.org

Desert Bus Bunker

During Desert Bus For Hope itself, we relocate to the "bunker", so as to avoid adding strain to the Desert Bus servers. You can find it over at irc.chatspike.net #DesertBusBunker.

Twitch Chat

The Twitch chat is where the action happens during the event itself, to talk with the crew on the stream and others watching along with you.
